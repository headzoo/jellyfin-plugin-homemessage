What is HomeMessage?

HomeMessage injects a small script into Jellyfin's web UI to show site-wide announcements on the home screen. Messages are dismissible per user and can be scheduled or styled.

Note: Index page injection requires write access to the active web directory. If permissions are restricted (common in Docker), point Jellyfin to a writable --webdir or see the README for alternatives.
